# 🎮 Simon Says Game

A simple Simon Says Game built using **HTML, CSS, and JavaScript**. The player has to remember and repeat the sequence of colors generated by the game. The sequence becomes longer with each level, making the game more challenging.

## Features

* Random color sequence generation
* Level progression
* User input validation
* Highest score tracking
* Game over and restart functionality

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Clone the repository.
2. Open `index.html` in your browser.
3. Press any key to start playing.
